Description

hoyo is a command-line utility that lets the user save directories as bookmarks (similar to in the browser) and easily cd to them.

data HoYoMonad a #

HoYoMonad is the main monad stack for the hoyo program. It's essentially a wrapper around ExceptT String (ReaderT Env IO): in other words, HoYoMonad a is equivalent to Env -> IO (Either String a)
