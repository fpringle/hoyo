newtype HoYoMonad a #

HoYoMonad is the main monad stack for the hoyo program. It's essentially a wrapper around ExceptT T.Text (ReaderT Env IO): in other words, HoYoMonad a is equivalent to Env -> IO (Either T.Text a)
